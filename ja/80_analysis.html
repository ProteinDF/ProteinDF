10. ProteinDF計算結果の解析

10.1. 一般利用向け解析ツール

  • pdf-component

分子軌道における基底関数(原子軌道)の寄与率を表示します。

Usage: pdf-component [Option]

コマンドを実行すると以下のような出力が得られます。

$ pdf-component 115
MO level: 115
No. 1     8  C (  42.525,   37.241,   32.674) p(    px) (  8.18%;   8.18%)
No. 2     8  C (  42.525,   37.241,   32.674) p(    pz) (  7.77%;  15.95%)
No. 3    20  C (  43.858,   35.420,   31.228) p(    px) (  7.38%;  23.34%)
No. 4    11  N (  41.507,   35.639,   33.859) p(    px) (  6.74%;  30.08%)
No. 5    14  C (  42.553,   33.543,   32.877) p(    px) (  6.69%;  36.77%)
No. 6    20  C (  43.858,   35.420,   31.228) p(    pz) (  5.54%;  42.31%)
No. 7    14  C (  42.553,   33.543,   32.877) p(    pz) (  5.20%;  47.51%)
No. 8    11  N (  41.507,   35.639,   33.859) p(    pz) (  4.71%;  52.23%)
No. 9     9  C (  41.623,   36.994,   33.649) p(    px) (  4.39%;  56.62%)
No.10     9  C (  41.623,   36.994,   33.649) p(    pz) (  3.39%;  60.00%)

  • pdf-kwd

ProteinDF入力ファイルにおけるキーワードを出力します。いくつかの形式で出力できます。詳しくはヘルプ(-h)を参照してください

Usage: pdf-kwd

  • pdf-pop-mulliken

Mulliken電荷密度分布を出力します。

Usage: pdf-pop-mulliken <MO_ID>

  • pdf-mkfld-mo

分子軌道のボリュームデータを出力します。いくつかの形式で出力できます。詳しくはヘルプ(-h)を参照してください

Usage: pdf-mkfld-mo <MO_ID>

  • pdf-mkfld-esp

静電ポテンシャルのボリュームデータを出力します。いくつかの形式で出力できます。詳しくはヘルプ(-h)を参照してください

Usage: pdf-mkfld-esp

  • pdf-mkfld-dens

電子密度のボリュームデータを出力します。いくつかの形式で出力できます。詳しくはヘルプ(-h)を参照してください

Usage: pdf-mkfld-dens

  • pdf-archive

計算結果のアーカイブを作成して集計します。

Usage: pdf-archive

  • pdf-report

計算結果の簡単なレポートを出力します。

Usage: pdf-archive

エネルギー収束カーブおよび分子軌道エネルギー収束履歴が得られます。

10.2. 応用(開発者向け)ツール

  • pdf-mat-show

バイナリ形式の行列ファイルをテキスト形式で出力します。

Usage: pdf-mat-show <matrix file>

  • pdf-vtr-show

バイナリ形式のベクトルファイルをテキスト形式で出力します。

Usage: pdf-vtr-show

  • mpac2yml.py

MessagePack形式のファイルをYAML形式に変換します。

Usage: mpac2yml.py [options] FILE

Options:
-h, --help  show this help message and exit

  • yml2mpac.py

YAML形式のファイルをMessagePack形式に変換します。

Usage: yml2mpac.py [options] YAML_FILE MPAC_FILE

Options:
-h, --help  show this help message and exit

  • pdf-diff

2つの計算結果を比較します。

Usage: pdf-diff

  • pdf-mat-info

行列ファイルの情報(行数、列数など)を出力します。

Usage: pdf-mat-info
